# NWO Robotics for ChatGPT

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Control robots, IoT devices, and robot swarms through ChatGPT using the Model Context Protocol (MCP).

## Overview

NWO Robotics brings powerful robotics control to ChatGPT. This MCP server enables natural language interaction with robots, IoT devices, and coordinated robot swarms.

## Features

- **Robot Control**: Direct control of individual robots
- **VLA Inference**: Visual-Language-Action model integration
- **Task Planning**: AI-powered task decomposition and execution
- **Sensor Fusion**: Multi-sensor data integration
- **Swarm Coordination**: Multi-robot collaboration
- **IoT Integration**: Smart device control
- **Safety Systems**: Built-in safety checks and emergency stops

## Installation

No installation required! This is a hosted MCP server that works directly within ChatGPT once approved.

## Usage

Once the app is approved by OpenAI, simply:

1. Open ChatGPT
2. Look for "NWO Robotics" in the GPTs/apps section
3. Start controlling robots with natural language!

Example commands:
- "Move the robot forward 2 meters"
- "Check the status of all IoT devices"
- "Deploy a swarm to patrol the perimeter"
- "Plan a task to pick up objects from the table"

## API

The MCP server exposes 60+ tools across these categories:

| Category | Tools | Description |
|----------|-------|-------------|
| Core Robotics | 15 | Robot control, VLA, planning, sensors |
| Agent Discovery | 5 | Health, capabilities, dry-run |
| Simulation | 6 | Trajectory, collision, grasp |
| Embodiment | 6 | Robot specs, URDF |
| Swarm | 3 | Multi-robot coordination |
| Tasks | 2 | Task management |
| IoT | 2 | Device control |
| Safety | 2 | Safety checks, emergency stop |
| Models | 4 | Model management |
